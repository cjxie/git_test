**InitRepo**

  git init main
  git add .
  git commit -m 'some commits'

**CloneRepo**

  git clone someRepo


**Config**

  Set the user info for code submission
  
  git config --global user.name "yourname"
  git config --global user.email "youremail@gmail.com"
